
# Software Requirements Specification (SRS)
## Smart Media Studio Nakuru

### 1. Introduction
This SRS defines the functional and non-functional requirements for the Smart Media Studio Nakuru Integrated Website and Photo Studio Management System.

### 2. System Overview
The system is a customer-facing website and a full backend management platform built with ASP.NET Core Razor Pages, Tailwind CSS, and MySQL.

### 3. Visual Identity
- Primary Color: #fcf4eb
- Accent Color: #eba536
- Light and Dark theme support

### 4. User Categories
- Management
- Staff
- Registered Customers
- Walking Customers
- Public Visitors

### 5. Core Functional Requirements
- Public website with 3D-inspired hero section
- Services catalog with pricing
- Photo & video galleries (managed by staff)
- Blogs and articles
- Testimonials with approval & expiry
- Staff profiles page
- Contact, inquiries, and live chat

### 6. Customer Self-Service Portal
- Registration & profile management
- Online ordering
- Job tracking
- Invoice & receipt viewing
- Feedback and communication

### 7. Online Payments
- M-Pesa (STK Push, Till/Paybill)
- Card payments (Visa, Mastercard)
- Payment confirmation & reconciliation

### 8. Non-Functional Requirements
- Accessibility (WCAG 2.1)
- SEO optimized
- Secure (HTTPS, CSRF, XSS protection)
- Scalable and performant
